Type & Code

Typography | Programming

[ Published on Type on Screen ]

Using Processing, I created a series of typographical experiments on forms and meanings.

Firstly, I created this set of letters based on the struture of a binary tree. Every letter has binary trees branching out, and the anchor points of the binary trees start from Chinese characters whose ponounciations begin with the corresponding letters in front of them. This probject is meant to address the idea of abstraction and obscurity.

Secondly, I did a series of form & structure studies using the letter i. The goal is experiment with a simple letterform and see how much depth and space it is able to demenstrate.
