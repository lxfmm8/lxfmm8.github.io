Markers UI

User Interface

Director: Fabian Borlein
Tools: InDesign, Illustrator
Year: 2015

In the process of creating mockups and digital resource gadgets for this complex enterprise software, I started by following along with a given standard style guide, including fonts, color palettes, and logic of the imagined user experience flow, and then I created a number of sets of user interfaces based on the goal of each page of logics (meetings, roadshows, contacts, feedback, alerts, etc.) and discussed with the team to come up with the better ones. At the same time, I had to consider specific styles for typography and layout strustures including how the UIs animate and handed them to the developer.
